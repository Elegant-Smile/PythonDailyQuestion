#### 基础题1：

```
用户输入一个字符串，修改该字符串中哪个位置的字符，程序就会输出修改后的结果。
比如用户输入 ：
字符串>> fkjava.org
字符位置>> 6
替换字符>> -
程序将会输出： 
替换后的字符串>> fkjava-org
```



#### 基础题2：

```
用户输入一个字符串和 一个子串，
程序必须打印出给定子串在目标字符串中出现的次数 。
字符串遍历将从左到右进行，而不是从右到左 。 
例如给定'ABCDCDC’和’CDC'，程序输出“ 2 ”。
```
